# simple-http-server

simple http server

## Dependencies

[golang]( https://golang.org/ )

## Usage

```sh
go run simple-http-server.go --doc-root /path/to/document-root --port 8080
```

or

```sh
go build simple-http-server.go
./simple-http-server --doc-root /path/to/document-root --port 8080
```
