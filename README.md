# oneline-http-server

simple oneline http server

## Usage

```sh
go run oneline-http-server.go --doc-root /path/to/document-root --port 8080
```

or

```sh
go build oneline-http-server.go
./oneline-http-server --doc-root /path/to/document-root --port 8080
```
